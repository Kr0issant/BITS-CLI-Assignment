### Question 5: Storage Health Assessment and Documentation [4 Marks]

**Scenario:** Your organization plans to deploy a new application server. Assess the storage resources and document your findings using the `vi` editor.

Collect storage information and prepare a report containing:

* Available storage devices.
* Mounted file systems.
* Disk usage statistics.
* Inode utilization.
* Storage health observations.
* Recommendations for improving storage utilization.

* `Storage_Assessment_Report.txt` *(Must be created explicitly using the vi editor)*.
* Screenshots showing `vi` editor operations.
